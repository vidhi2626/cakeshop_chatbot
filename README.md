# 🎂 Cake Shop Chatbot

## 📌 Overview

Cake Shop Chatbot is a rule-based chatbot developed using Python. It helps customers by answering common queries related to cakes, pricing, shop timings, location, recommendations, and custom cake orders.
The chatbot uses predefined rules and keyword matching to provide quick and interactive responses.

---

## 🎯 Features

* Greeting and basic conversation support
* Cake menu information
* Cake price details
* Shop location and timings
* Cake recommendations based on occasions
* Custom cake inquiry support
* Exit conversation with farewell messages

---

## 🛠️ Technologies Used

* Python
* Dictionary-based Rule Engine
* String Matching
* Command Line Interface (CLI)

---

## 🚀 How to Run

### Step 1: Clone Repository

```bash
git clone https://github.com/vidhi2626/cakeshop_chatbot.git
```

### Step 2: Navigate to Project Folder

```bash
cd cakeshop_chatbot
```

### Step 3: Run Chatbot

```bash
python cakeshop.py
```

## 💬 Sample Conversation

```text
You: hello
Bot: Welcome to the Happy Cake Shop! How can I help you today?

You: menu
Bot: We have Chocolate Truffle, Red Velvet, Pineapple Cream, and Black Forest cakes.

You: chocolate cake
Bot: Our Chocolate Truffle cake is a rich, decadent classic. It costs ₹600.

You: bye
Bot: Thank you for visiting! We hope to see you again soon.
Have a good day!
```
